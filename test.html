Problem 13a) Verify (A △ B) ∪ C = (A ∪ C) △ (B \ C)

Prerequisite notation:

Set Difference: B \ C = B ∩¬C
Symmetric Difference: A △ B = (A ∪ B) \ (A ∩ B) or A △ B = (A \ B) ∪ (B \ A)

Start by working backwards from
(A ∪ C) △ (B \ C) = ((A ∪ C) \ (B \ C)) ∪ ((B \ C) \ (A ∪ C))
((A ∪ C) \ (B \ C)) ∪ ((B \ C) \ (A ∪ C)) = ((A ∪ C) ∩¬(B ∩¬C)) ∪ (((B ∩¬C) ∩¬(A ∪ C))
Switch to logic symbols and apply DeMorgan’s Law where applicable
((A ∨ C) ∧¬(B ∧¬C)) ∨ ((B ∧¬C) ∧¬(A ∨ C)) = ((A ∨ C) ∧ (¬B ∨ C)) ∨ ((B ∧¬C) ∧ (¬A ∧¬C))
Apply associvativty law to right most equation (can drop the paranthesis). Apply Idempotence to ¬C
((A ∨ C) ∧ (¬B ∨ C)) ∨ ((B ∧¬C) ∧ (¬A ∧¬C)) = ((A ∨ C) ∧ (¬B ∨ C)) ∨ (B ∧¬C ∧¬A)
Distrubute (A ∨C) over (¬B ∨C)
((A ∨ C) ∧ (¬B ∨ C)) ∨ (B ∧¬C ∧¬A) = (((A ∨ C) ∧¬B) ∨ ((A ∨ C) ∧ C)) ∨ (B ∧¬C ∧¬A)
Apply absorption to ((A ∨C) ∧C))
(((A ∨ C) ∧¬B) ∨ ((A ∨ C) ∧ C)) ∨ (B ∧¬C ∧¬A) = (((A ∨ C) ∧¬B) ∨ C) ∨ (B ∧¬C ∧¬A)
Distribute ¬Bover(A ∨C)
(((A ∨ C) ∧¬B) ∨ C) ∨ (B ∧¬C ∧¬A) = ((A ∧¬B) ∨ (C ∧¬B) ∨ C) ∨ (B ∧¬C ∧¬A)
Apply absorption to (C ∧¬B) ∨C
((A ∧¬B) ∨ (C ∧¬B) ∨ C) ∨ (B ∧¬C ∧¬A) = ((A ∧¬B) ∨ C) ∨ (B ∧¬C ∧¬A)
Use associative law to move around paranthesis. Distrube C over (B ∧¬C ∧¬A)
(A ∧¬B) ∨ C ∨ (B ∧¬C ∧¬A) = (A ∧¬B) ∨ ((B ∨ C) ∧ (¬C ∨ C) ∧ (¬A ∨ C))
(A ∧¬B) ∨ ((B ∨ C) ∧ (¬C ∨ C) ∧ (¬A ∨ C)) = (A ∧¬B) ∨ ((B ∨ C) ∧ (¬A ∨ C))
Use distrubtion law to move C from ((B ∨C) ∧(¬A ∨C))
(A ∧¬B) ∨ ((B ∨ C) ∧ (¬A ∨ C)) = (A ∧¬B) ∨ (B ∧¬A) ∨ C
Switching back to set notation.
(A ∩¬B) ∪ (B ∩¬A) ∪ C = (A \ B) ∪ (B \ A) ∪ C = (A △ B) ∪ C
Using the definition for set and symmetric difference, we have verified that
(A ∪ C) △ (B \ C) = (A △ B) ∪ C
